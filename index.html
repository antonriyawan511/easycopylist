    Info & Cara Penggunaan

    Membuat Grup Baru

    Masukkan teks item dipisahkan dengan simbol "~" di textarea utama, lalu klik "Save". Grup akan dibuat otomatis dengan nama berdasarkan kata-kata umum.

    Mengganti Grup

    Klik ikon menu di kiri atas, lalu pilih grup dari daftar.

    Menyalin Item

    Klik pada item untuk menyalin teksnya ke clipboard . Jumlah salin akan tercatat.

    Melihat Teks Lengkap

    Tekan lama (long press) pada item untuk melihat teks lengkap dalam modal .

    Menambah Variasi

    Klik tombol "Add Variasi" pada item untuk menambah variasi teks.

    Melihat Daftar Variasi

    Klik tombol "List Variasi" untuk melihat dropdown variasi. Klik pada variasi untuk menyalin, atau tekan lama untuk melihat lengkap.

    Mengedit Item/Variasi

    Klik tombol pada item, lalu pilih "Edit". Untuk variasi, klik ikon di dropdown.

    Menghapus Item/Variasi

    Dari menu atau dari dropdown variasi, pilih "Delete" .

    Mereset Jumlah Salin

    Klik ikon reset di header grup untuk mengatur ulang counter salin.

    Mengganti Nama Grup

    Klik ikon pensil di header grup.

    Export Grup

    Klik ikon download di header grup untuk export sebagai file .txt.

    Menghapus Grup

    Klik ikon sampah di header grup.

    Edit Main Prompt

    Klik "Edit Main Prompt" untuk mengubah prompt utama yang digunakan untuk generate teks.

    Copy & Open Gemini

    Klik untuk menyalin prompt utama dan membuka Gemini di tab baru .

    Import/Export Semua Data

    Dari menu samping, klik ikon import untuk memuat data dari file .json, atau export untuk menyimpan semua data.

    Tampilan Responsif

    Aplikasi menyesuaikan tampilan untuk desktop, tablet, dan mobile.

    Penyimpanan Lokal

    Semua data disimpan di browser localStorage, tidak memerlukan server.
